Privacidade

Comunicação inteligente para o seu negócio

1. Conformidade com a LGPD

O tizap! está em total conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018). Esta política descreve como tratamos dados pessoais no contexto de nossa ferramenta de automação de WhatsApp.

2. Coleta de Dados

Coletamos apenas os dados estritamente necessários para a prestação de nossos serviços de mensageria:

  • Dados de cadastro (E-mail e Senha)
  • Números de telefone dos destinatários das mensagens
  • Conteúdo das mensagens e fluxos automatizados
  • Logs de interação para fins de suporte e melhoria contínua

3. Finalidade do Tratamento

Os dados são utilizados exclusivamente para as finalidades permitidas pela LGPD:

  • Execução de Contrato: Para possibilitar o disparo das comunicações contratadas.
  • Legítimo Interesse: Para segurança da plataforma e prevenção de fraudes.
  • Consentimento: Quando necessário para comunicações de marketing do próprio sistema.

4. Direitos do Titular

De acordo com a LGPD, os titulares dos dados têm direito a:

  • Confirmação da existência de tratamento.
  • Acesso aos dados e correção de dados incompletos ou inexatos.
  • Anonimização, bloqueio ou eliminação de dados desnecessários.
  • Portabilidade dos dados a outro fornecedor de serviço.
  • Revogação do consentimento a qualquer momento.

5. Segurança e Armazenamento

Utilizamos medidas técnicas e administrativas avançadas para proteger os dados pessoais de acessos não autorizados e de situações acidentais ou ilícitas de destruição, perda ou alteração. Os dados são armazenados em servidores seguros com criptografia de ponta.

Importante: O tizap! utiliza a WhatsApp Business API oficial, garantindo que o tráfego de mensagens siga os mais altos padrões de segurança e privacidade da Meta.
